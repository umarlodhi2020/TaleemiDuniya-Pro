### 🚀 1. How to Access the Software (`Software Khulna Kese Hai?`)
You can open TaleemiDunya-Pro on **any device** (Android Phone, iPhone, iPad, Windows Laptop, or Mac Desktop) anywhere in the world!
* **Step 1:** Open Google Chrome or Safari browser on your device.
* **Step 2:** Type the live official school link in the address bar:
  👉 **`https://taleemidunya-pro-ed44e.web.app`**
* **Step 3:** You will see the glowing welcome login screen!

---

### 🔑 2. Understanding Role-Based Login (`4 Rango Ke Logins`)
TaleemiDunya-Pro has built-in security that automatically gives right access based on who is logging in:

1. **👨‍🏫 Principal / School Admin (`Master Role`)**
   * Can see and manage everything: Students, Staff Salaries, Fee Challans, Gate Pass Scanners, and Cloud Backups.
2. **👩‍🏫 Teachers (`Classroom Role`)**
   * Can only see assigned classes, mark daily student attendance, enter exam marks, and write daily homework diary.
3. **👪 Parents (`Family Portal`)**
   * Can only see their own children's attendance, pending fee challans, exam results, and daily diary.
4. **👑 Super Admin (`SaaS Owner - Umar Bhai`)**
   * Controls global licenses, school subscriptions, and cloud infrastructure across multiple campuses.

### 📲 3. Installing the Native Android App (`📲 Install App Button`)
You don't need to open the browser every day! You can install TaleemiDunya-Pro as a permanent Home Screen App:
1. Click the green button at the top right of the dashboard: **`📲 Install App`**.
2. A window will open. If you have an Android phone or tablet, click the button:
   👉 **`📦 Download Android APK App (`v2.5.0.apk`)`**
3. Open the downloaded `.apk` file and tap **Install** to get the permanent school icon right next to your WhatsApp and Camera icons!
