# 👑 TALEEMI DUNYA PRO — PRINCIPAL & STAFF TRAINING LECTURE 05
## 🌟 Academic Operations, Daily Homework Diary & Online Quiz Engine (`Parihai, Timetable Aur Exams`)

In **Lecture 05**, we dive into the day-to-day academic operations: how teachers post daily homework (`Daily Diary`), how to build conflict-free class timetables, and how to conduct computerized online quizzes.

---

### 📓 1. Daily Homework Diary (`/school-admin/daily-diary`)
Parents love checking what homework their children received in school every day! Teachers can post homework from their phones in seconds:
1. Open **`Left Sidebar -> Academic Hub -> Daily Homework Diary`**.
2. Select your Class (`e.g., Class 10`) and Subject (`Math, Science, English, Urdu`).
3. Type the homework details (`e.g., Exercise 4.2 all questions to be solved on notebook + Chapter 5 reading`).
4. Click **`📢 Publish Daily Diary`**.
   * *Instant Impact:* As soon as published, the homework appears live on the **Parent & Student Mobile App**, ensuring parents stay 100% engaged with their child's education!

---

### 📅 2. Class Timetable Builder (`/school-admin/timetable-builder`)
Creating class routines and allocating periods to teachers without clashes is super easy:
1. Open **`Left Sidebar -> Academic Hub -> Timetable Builder`**.
2. Select target Class (`Class 10`).
3. Click **`➕ Add Period / Slot`** for each time block (`e.g., Period 1: 08:30 AM - 09:15 AM`).
4. Assign Subject (`Mathematics`), Teacher (`Sir Imran Ali`), and Room Number (`Room 102`).
5. Click **`🖨️ Print Class Routine`** to generate a clean, colorful PDF schedule to paste inside classrooms!

---

### 📝 3. Exam Results & Report Card Generator (`/school-admin/exams`)
When terminal examinations or monthly tests conclude, generating official report cards takes only minutes:
1. Open **`Left Sidebar -> Academic Hub -> Examination Engine`**.
2. Create your Exam Term (`e.g., First Term Examination 2026`).
3. Enter marks obtained against total marks (`e.g., Math: 88/100, Physics: 92/100, English: 78/100`).
4. Click **`🖨️ Print Report Cards`**:
   * The system automatically calculates total percentage, letter grade (`A+, A, B`), class rank, and prints formal school-branded report cards complete with Principal signature lines!

---

### 💻 4. Online Computerized Quiz Engine (`/school-admin/quiz-engine`)
If your school has a computer lab or tablet class, you can conduct live digital tests:
* Open **`Academic Hub -> Online Quiz Engine`**.
* Create multiple-choice questions (`MCQs`) with 4 options and set the correct answer.
* Students log into their portal -> tap **`Start Quiz`** -> answer questions within a live countdown timer (`15 minutes`) -> get instant AI scores!
